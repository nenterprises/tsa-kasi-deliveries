# Customer UI/UX Visual Guide

## Color Scheme

### Primary Colors
- **Blue:** `bg-blue-600` - Primary actions (buttons, links)
- **Purple:** `bg-purple-600` - Custom requests, secondary actions
- **Green:** `bg-green-600` - Success states, checkout
- **Red:** `bg-red-600` - Errors, removals, warnings

### Status Colors
- **Yellow:** Pending orders
- **Blue:** Received by agent
- **Purple:** Items purchased
- **Indigo:** On the way
- **Green:** Delivered
- **Red:** Cancelled

## Components & Elements

### Buttons
- **Primary:** Blue background, white text
- **Success:** Green background, white text
- **Secondary:** Purple background, white text
- **Danger:** Red text or background
- **Disabled:** 50% opacity, no pointer events

### Cards
- White background
- Rounded corners (`rounded-lg`)
- Shadow (`shadow-md`)
- Hover effect (`hover:shadow-xl`)

### Forms
- Labeled inputs with required indicators (*)
- Border with focus ring on focus
- Error messages in red boxes
- Success messages in green boxes

### Status Badges
- Rounded pill shape
- Colored background with border
- Icon + text

### Navigation
- Sticky header on scroll
- Back button on detail pages
- Cart badge with count
- Logout in header

## Responsive Design

### Mobile (< 640px)
- Single column layouts
- Stacked navigation
- Full-width cards
- Simplified headers

### Tablet (640px - 1024px)
- 2-column product grids
- Side-by-side forms
